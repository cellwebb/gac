# XML tags of structuur te gebruiken - schrijf gewoon platte instructies voor de AI.
#

# 1. Kopieer dit bestand en pas het aan naar je voorkeuren
# 2. Stel GAC_SYSTEM_PROMPT_PATH=/path/to/your/custom/prompt.txt in .gac.env in
#
# De AI ontvangt je aangepaste instructies samen met de git diff gegevens.
